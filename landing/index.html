FAQ

The important questions.

Is this legal?

Modifying the speed limit voids the scooter's type approval (ABE) and insurance coverage. Operating a modified scooter on public roads may be illegal in your jurisdiction. FlashMonkey is intended for private property and research purposes only.

Can I go back to stock firmware?

Yes. FlashMonkey includes a one-click rollback that restores the original firmware. Your scooter will be exactly as it was before.

Will Navee firmware updates break it?

If Navee pushes a firmware update via their app, it will overwrite the patch. FlashMonkey re-patches automatically using the same license key. The patch engine finds the correct location in any firmware version using pattern matching.

Is my scooter safe?

The patch changes exactly 2 bytes of executable code (a conditional branch becomes a no-op). The SHA-256 integrity checksum is recalculated correctly, so the scooter's bootloader treats the firmware as genuine. The rest of the firmware is completely untouched.

What if it doesn't work?

14-day money-back guarantee. If FlashMonkey can't unlock your scooter for any reason, you get a full refund. No questions asked.

How does the license key work?

Your key is bound to your scooter's Bluetooth MAC address on first use. It works forever on that specific scooter, including re-patches after Navee updates. One key = one scooter.
