# Microcontroller — ESP32 / Arduino

Dieses Verzeichnis ist fuer zukuenftige Microcontroller-Implementierungen vorgesehen, die direkt mit dem Navee ST3 Pro ueber BLE kommunizieren.

## Geplante Projekte

- **ESP32 BLE Controller** — Standalone-Controller auf ESP32-Basis zur Steuerung des Scooters ohne Smartphone
- **ESP32 Telemetrie-Logger** — Echtzeit-Logging von Geschwindigkeit, Akku und Temperatur auf SD-Karte oder Display
- **Arduino Dashboard** — OLED/TFT-Display mit Echtzeit-Anzeige der Scooter-Telemetrie
