# Reverse Engineering — Tools & Skripte

Dieses Verzeichnis ist fuer Tools und Skripte zur Protokoll-Analyse des Navee ST3 Pro vorgesehen.

## Geplante Inhalte

- **BLE-Packet-Parser** — Python-Skript zum Parsen von BT-HCI-Captures
- **Frame-Builder** — Hilfstool zum Erstellen von BLE-Frames mit korrekter Checksum
- **Wireshark-Dissector** — Lua-Plugin fuer Wireshark zur Anzeige der Navee-Protokollfelder
- **APK-Analyse-Notizen** — Annotierte Auszuege aus der dekompilierten APK
